# 供应链质量智能分析平台

## 项目概述
这是一个基于Streamlit构建的供应链质量管理分析平台，专门用于PCB制造行业的质量数据分析和监控。

## 主要功能

### 1. 智能筛选系统
- **时间范围选择**：灵活选择分析时间段
- **供应商多选**：支持多家供应商对比分析
- **质量特性筛选**：针对不同质量参数进行专项分析
- **工序筛选**：按不同生产工序查看质量数据

### 2. 核心指标与预警
- **KPI概览面板**：实时显示关键质量指标
- **CPK智能计算**：自动计算过程能力指数
- **风险预警系统**：
  - CPK < 1.33：红色警告，制程能力不足
  - CPK ≥ 1.33：绿色提示，制程能力良好
- **供应商专项预警**：识别具体供应商的质量风险

### 3. 计量型数据分析
- **趋势控制图**：
  - 时间序列质量趋势展示
  - 规格中心线、上下限线标注
  - 超规数据点高亮标记
  - 多供应商对比显示
- **过程能力分析**：
  - 测量值分布直方图
  - 正态分布拟合曲线
  - CPK值可视化展示
- **供应商对比分析**：
  - 箱线图展示各供应商数据离散程度
  - 统计指标对比

### 4. 计数型数据分析
- **良率趋势监控**：
  - 合格率时间趋势图
  - 目标良率参照线
  - 多供应商良率对比
- **不良分析工具**：
  - 帕累托图（80/20法则）
  - 不良原因频次统计
  - 累计百分比分析

### 5. 数据导出功能
- **异常数据一键导出**：
  - 超规测量数据
  - 低良率批次数据
  - Excel格式报告生成

## 技术架构

### 前端框架
- **Streamlit**：快速构建数据应用
- **Plotly**：交互式图表可视化
- **自定义CSS**：企业级UI美化

### 数据处理
- **Pandas**：数据清洗和分析
- **NumPy**：数值计算
- **统计分析**：CPK计算、分布分析

### 文件处理
- **OpenPyXL**：Excel文件读写
- **XlsxWriter**：报表生成

## 安装部署

### 环境要求
- Python 3.7+
- pip包管理器

### 安装步骤
```bash
# 克隆项目
git clone <repository-url>
cd supply_chain_quality_platform

# 安装依赖
pip install -r requirements.txt

# 启动应用
streamlit run supply_chain_quality_app.py
```

### 访问应用
应用启动后，在浏览器中访问：`http://localhost:8501`

## 使用指南

### 1. 数据导入
- 支持Excel文件(.xlsx)格式
- 文件应包含计量型和计数型两个数据表
- 或使用内置示例数据进行演示

### 2. 数据筛选
- 在左侧边栏设置筛选条件
- 实时更新主面板分析结果
- 支持多维度组合筛选

### 3. 质量分析
- 查看KPI概览了解整体质量状况
- 通过计量型分析深入探究具体质量问题
- 利用计数型分析优化生产工艺

### 4. 异常处理
- 关注红色预警信息
- 导出异常数据进行进一步分析
- 跟踪改进措施效果
